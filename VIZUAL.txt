✨ ВИЗУАЛЬНЫЕ ЭФФЕКТЫ:

1. VHS SCANLINES - горизонтальные линии по всему экрану
   Создают эффект старого VHS плеера

2. ГЛИТЧ НА ЗАГОЛОВКЕ - RGB разделение, дрожание
   Текст "разваливается" на красный/зелёный/синий

3. ГРАДИЕНТНЫЙ ФОН - фиолетово-синий космос
   linear-gradient от тёмно-фиолетового к чёрному

4. ИКОНКИ ДИСКЕТ - чисто CSS (без картинок!)
   - Прямоугольник с закруглёнными углами
   - Тёмная полоска сверху (этикетка)
   - Круг внизу (отверстие в центре)
   - Cyan градиент + свечение

5. HOVER ЭФФЕКТЫ:
   - Дискета поднимается вверх
   - Усиливается свечение (glow)
   - Лёгкий поворот (rotate)
   - Кнопки масштабируются


🎮 ИНТЕРАКТИВ:

- Клик на доступную дискету → переход на страницу чтения
- Hover на дискете → анимация + подсветка
- Недоступные дискеты серые, не кликабельны
- Кнопки с "shine" эффектом (блик при наведении)


🎨 ЦВЕТОВАЯ СХЕМА:

ОСНОВНЫЕ ЦВЕТА:
- Фон: #1a0033 (тёмно-фиолетовый) → #0d001a (почти чёрный)
- Текст: #00ffff (cyan) и #ff00ff (magenta)
- Акценты: #00ff00 (зелёный для счётчика)

ГРАДИЕНТЫ:
- Дискеты: cyan (#00ffff) → синий (#0099ff)
- Кнопки: cyan (#00ffff) → magenta (#ff00ff)

ЭФФЕКТЫ СВЕЧЕНИЯ (box-shadow):
- Cyan glow: 0 0 20px rgba(0, 255, 255, 0.5)
- Magenta glow: 0 0 20px rgba(255, 0, 255, 0.5)
