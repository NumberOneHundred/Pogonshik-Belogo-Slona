Вот как должна выглядеть твоя папка с сайтом:

pogonschik-slona/
│
├── index.html              ← Главная страница
├── style.css               ← Стили (цвета, шрифты)
│
├── disketa-1.html          ← Страница дискеты 1
├── disketa-2.html          ← Страница дискеты 2
├── disketa-3.html          ← Страница дискеты 3
├── ...                     (создай 20 таких файлов)
├── disketa-20.html         ← Страница дискеты 20
│
├── vse-diskety.html        ← Список всех дискет (опционально)
│
├── files/                  ← СОЗДАЙ ЭТУ ПАПКУ!
│   ├── disketa-1.pdf       ← Твой PDF с дискетой 1
│   ├── disketa-2.pdf       ← Твой PDF с дискетой 2
│   ├── disketa-3.pdf
│   ├── ...
│   ├── disketa-20.pdf
│   └── vse-diskety.pdf     ← (Опционально) Все дискеты в одном PDF
│
├── README.txt              ← Инструкция
└── STRUKTURA.txt           ← Этот файл


═══════════════════════════════════════════════════════════════
  ЧТО СДЕЛАТЬ:
═══════════════════════════════════════════════════════════════

1. Создай папку "files"
2. Положи туда все свои PDF с дискетами
3. Назови их: disketa-1.pdf, disketa-2.pdf, ... disketa-20.pdf
4. Скопируй disketa-1.html и создай ещё 19 копий (disketa-2.html ... disketa-20.html)
5. В каждой копии замени номера (см. README.txt)
6. Загрузи всё на GitHub Pages или Neocities


═══════════════════════════════════════════════════════════════
  ВАЖНО!
═══════════════════════════════════════════════════════════════

- Названия файлов должны быть БЕЗ ПРОБЕЛОВ
- Правильно: disketa-1.pdf
- Неправильно: дискета 1.pdf

- PDF файлы должны быть в папке "files"
- Если PDF не в папке "files", ссылки не будут работать!
