📁 ЧТО ВНУТРИ:

1. index.html - главная страница
2. disketa-1.html ... disketa-20.html - ВСЕ 20 СТРАНИЦ

4. pdf-viewer.js - скрипт для отображения PDF

📋 ЧТО ДЕЛАТЬ:

ШАГ 1: Подготовь PDF
─────────────────────────────────────────────────────────────
1. Создай папку "files"
2. Положи туда: disketa-1.pdf, disketa-2.pdf ... disketa-20.pdf


ШАГ 2: Замени ссылки и имена
─────────────────────────────────────────────────────────────
Открой ЛЮБОЙ текстовый редактор с функцией "найти и заменить":

1. Замени во ВСЕХ файлах:
   "https://t.me/ВАШ_КАНАЛ" → твой Telegram
   "[ВАШЕ ИМЯ ФАМИЛИЯ]" → твоё имя
   "[ИМЯ ФАМИЛИЯ ХУДОЖНИЦЫ]" → имя художницы


ШАГ 3: Загрузи на GitHub
─────────────────────────────────────────────────────────────
1. github.com → New repository → "pogonschik-slona"
2. Загрузи ВСЕ файлы (включая pdf-viewer.js!)
3. Settings → Pages → Source: main → Save
4. Готово! Сайт: https://твой-ник.github.io/pogonschik-slona/

  КАК РАБОТАЕТ PDF VIEWER:
