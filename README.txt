3. (Опционально) Создай общий PDF со всеми дискетами:
   - vse-diskety.pdf (для кнопки "Скачать PDF" на главной)


ШАГ 2: Замени ссылки
─────────────────────────────────────────────────────────────

Открой index.html и замени:

- "https://t.me/ВАШ_КАНАЛ" → ссылка на твой Telegram
- "https://www.litres.ru/ВАШ_ПРОФИЛЬ" → ссылка на Литрес

То же самое в disketa-1.html и disketa-2.html.


ШАГ 3: Создай страницы для всех дискет
─────────────────────────────────────────────────────────────

1. Скопируй файл disketa-2.html
2. Переименуй копию в disketa-3.html
3. Открой disketa-3.html и замени:
   - "ДИСКЕТА 2" → "ДИСКЕТА 3" (во всех местах!)
   - "disketa-2.pdf" → "disketa-3.pdf"
   - "ДИСКЕТА 1" → "ДИСКЕТА 2" (в кнопке назад)
   - "disketa-1.html" → "disketa-2.html" (в кнопке назад)
   - "ДИСКЕТА 3" → "ДИСКЕТА 4" (в кнопке вперёд)
   - "disketa-3.html" → "disketa-4.html" (в кнопке вперёд)

Повтори для всех дискет с 3 по 20.

ВАЖНО: Для последней дискеты (disketa-20.html) кнопка "вперёд" 
должна вести обратно на index.html с текстом "НА ГЛАВНУЮ ►"


ШАГ 4: Настрой счётчик дискет на главной
─────────────────────────────────────────────────────────────

В index.html найди:

<div class="counter">
    <p>◉ НАЙДЕНО: <span class="found-count">18</span> / <span class="total-count">20</span> ДИСКЕТ</p>
</div>

Меняй число "18" по мере того, как выпускаешь новые дискеты!


ШАГ 5: Включи/выключи дискеты
─────────────────────────────────────────────────────────────

Если хочешь, чтобы дискета была недоступна (показывалась как "СКОРО"):

В index.html замени:

<a href="disketa-6.html" class="disketa-card available">
    <div class="disketa-icon"></div>
    <div class="disketa-label">ДИСКЕТА 6</div>
</a>

На:

<div class="disketa-card unavailable">
    <div class="disketa-icon locked"></div>
    <div class="disketa-label">ДИСКЕТА 6<br><span class="soon">СКОРО</span></div>
</div>

Сейчас дискеты 1-18 доступны, 19-20 - недоступны.

ВАРИАНТ А: Через веб-интерфейс GitHub (без Git, самый простой)

1. Зарегистрируйся на github.com
2. Создай новый репозиторий:
   - Нажми "New repository"
   - Имя: "pogonschik-slona" (или любое другое)
   - Поставь галочку "Public"
   - Нажми "Create repository"

3. Загрузи файлы:

   - Перетащи все файлы (index.html, style.css, disketa-*.html)

4. Включи GitHub Pages:

   - Source: выбери "main" branch
   - Нажми "Save"

5. Готово! Твой сайт будет доступен по адресу:
   https://твой-ник.github.io/pogonschik-slona/

🎨 ИЗМЕНИТЬ ЦВЕТА:

Открой style.css и найди эти строки:

1. ФОНОВЫЙ ГРАДИЕНТ (строка ~12):
   background: linear-gradient(135deg, #1a0033 0%, #0d001a 50%, #1a0033 100%);
   
   Можешь заменить на:
   - Синий киберпанк: #000033, #000066
   - Красный Matrix: #1a0000, #330000
   - Зелёный хакер: #001a00, #003300

2. ЦВЕТ ТЕКСТА (строка ~15):
   color: #00ffff; (cyan)
   
   Можешь заменить на:
   - #ff00ff (magenta)
   - #00ff00 (green)
   - #ffffff (white)

3. ГРАДИЕНТЫ ДИСКЕТ (строка ~164):
   background: linear-gradient(135deg, #00ffff 0%, #0099ff 100%);
   
   Экспериментируй с разными цветами!

Q: PDF не показывается в браузере
A: Это нормально для некоторых браузеров. Появится кнопка 
   "скачать дискету" - пользователь просто скачает PDF.

Q: Как изменить, сколько дискет доступно?
A: Замени класс "available" на "unavailable" и добавь "locked" 
   к иконке + подпись "СКОРО" (см. ШАГ 5)

Q: Глитч-эффект слишком сильный
A: В style.css найди "@keyframes glitch" и уменьши значения 
   text-shadow (например, с 0.05em до 0.02em)

Q: Хочу свой домен (не github.io)
A: Купи домен на reg.ru (~300₽/год) и привяжи к GitHub Pages 
   (инструкции в настройках GitHub Pages)
